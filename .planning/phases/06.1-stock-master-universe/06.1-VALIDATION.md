## Sampling Rate

- **After every task commit:** Run quick command (해당 워크스페이스만 필터)
- **After every plan wave:** Run full suite command
- **Before `/gsd-verify-work`:** Full suite must be green + `bash scripts/smoke-master-sync.sh` PASS + 삼성전자 검색 E2E PASS
- **Max feedback latency:** 30 seconds (quick)
