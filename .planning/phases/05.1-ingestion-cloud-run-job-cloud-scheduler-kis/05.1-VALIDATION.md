- **After every task commit:** 스크립트 파일만 변경되는 경우가 대부분이므로 `bash -n scripts/*.sh`(구문 검사)만 실행

- **Before `/gsd-verify-work`:** `bash scripts/deploy-ingestion.sh` 전체 flow 통과
- **Max feedback latency:** smoke 60초 / 전체 배포 3분
