A

랭킹 행 — theme-rank-row 직계

바로 위 테마 섹션과 형제처럼 보이는 단일행 그리드(순위·종목·강도바·동반율·실시간). 가장 조밀·스캔 용이. 강도바 = --up(빨강) "상승 동반 강도". 모바일에선 강도바 열 숨김.

B

정보 카드 행 — 2줄 + 하단 강도바

근거를 전용 줄에 풀어 칩이 안 눌림. 1줄: 종목·실시간 / 2줄: 동반율·표본·근거칩·후행 배지 / 하단: 강도바. 강도바 = --primary(브랜드 블루) — "강도 ≠ 등락" 명확히. 정보량 최다.

C

지표 듀얼 — compact, 메트릭 우선

우측 듀얼 메트릭(동반율 大 + 실시간 小)에 시선 집중. 강도는 행 하단 얇은 라인(3px)으로 baseline 표시. 테이블 리듬에 가깝게 가장 빽빽. 근거·후행은 종목명 아래.

∅

빈 상태 — 두 경로 모두 빌 때만(D-06)

테마 없음 AND co-surge 이웃도 없음일 때. 약한 후보는 숨기지 않고 신뢰도 배지로 노출하므로(D-05) 이 상태는 드묾.

아직 이 종목과 함께 움직인 패턴을 찾지 못했습니다. 분류된 테마와 동반 급등 이력이 쌓이면 후보가 표시됩니다.

공통 규칙(전 변형 동일):
· 동반율(conf_d0)은 확률 → 항상 중립색(--fg), 절대 빨강/파랑 금지. 실시간 등락률만 상승=빨강/하락=파랑.
· 근거 = 공유 테마 칩(outline) 및/또는 "직접동반 N회"(accent fill) — 둘 다 해당이면 둘 다 표시(D-03).
· 후행형 배지 = D+1 후행 우세 후보(D-08). 정렬은 결합 점수, 그룹 분리 안 함.
· 강도바/라인 = 결합 점수(conf_d0+lift+avg_ret+conf_d1). lift는 내부 디노이즈(미표시).
· TOP-K 기본 8 + 더보기. 종목명/실시간 등락률은 stock_quotes 조인 실측.
