## Spacing Scale

Declared values (multiples of 4 — `globals.css` `--s-*` 토큰):

| Token | Value | Usage |
|-------|-------|-------|
| `--s-1` | 4px | (예약) 미세 간격 |
| `--s-2` | 8px | 근거 칩 row gap, 행 카드 간 리스트 gap, 더보기 버튼 상단 margin |
| `--s-3` | 12px | 행 카드 세로 패딩, 그리드 2열 gap, 메트릭(동반율↔실시간) gap |
| `--s-4` | 16px | 행 카드 가로 패딩 |
| `--s-5` | 24px | 빈 상태 박스 패딩(세로), 부모 `space-y-8` 사이 시각 호흡 |
| `--s-6` | 32px | (부모 컨텍스트) `space-y-8` 섹션 스택 간격 |

**Radius:** 행 카드 / 더보기 버튼 = `--r` (6px). 빈 상태 박스 = `--r-md` (8px). 칩·배지 = `rounded-full` (999px pill).

Exceptions (Dimension 5 통과용 — 모두 hairline 지표 또는 타이포 영역, 레이아웃 간격 아님):
- **1px** — 행 카드 / 빈 상태 / 더보기 버튼의 border (hairline).
- **3px** — 강도 라인(`.underbar`) 높이. 결합 점수를 baseline 으로 표현하는 hairline 지표(막대 아님).
- **5~6px** — 칩 내부 아이콘↔텍스트 gap. `theme-chips`의 `gap-1.5`(6px) 선례. 공유 테마 칩 도트 gap 5px, co-surge 칩 화살표 gap 4px(아이콘 광학 보정).
- **10~11px** — 미니라벨("동반율"/"실시간") 폰트 크기(10px) 및 후행/직접동반 배지 폰트(11px). 타이포 스케일 영역이며 spacing 토큰이 아님. `StockThemeChips` overflow 버튼이 동일 11px 선례.
- **62px** — 실시간 등락률 컬럼 `min-width`(고정폭 숫자 정렬 안정용 — theme-rank-row `min-w-[72px]` 동류 선례).

## Typography

`globals.css` `--t-*` 토큰 기반. 3개 핵심 사이즈 + 2개 보조 미니라벨. 본문 `--lh-normal` 1.5 / 타이트 `--lh-tight` 1.2.

| Role | Size | Weight | Line Height | Element |
|------|------|--------|-------------|---------|
| Heading (섹션 캡션) | `--t-caption` 12px | 600 (semibold) | 1.5 | `<h2>` "동조 후보" — uppercase tracking-wide, `--muted-fg` |
| Metric (동반율) | `--t-h3` 20px | 800 (extrabold) | 1.2 | 우측 동반율 `.mono`, 중립 `--fg` |
| Body / 종목명 | `--t-base` 16px | 700 (bold) | 1.5 | 종목명 `<b>` (`--fg`) |
| Live (실시간 등락률) | `--t-sm` 14px | 700 (bold) | 1.2 | 실시간 등락률 `.mono`, 등락 방향 색 |

**보조(미니라벨 / 칩·배지 — 타이포 영역):**
- 미니라벨 "동반율"/"실시간": 10px, weight 600, `--font-sans`, `--muted-fg`, letter-spacing 0.04em, 메트릭 위 3px.
- 근거 칩(공유 테마 / 직접동반): `--t-caption` 12px. 공유 테마 weight 400 / 직접동반 weight 600.
- 후행형 배지: 11px, weight 700.
- 종목코드: `--t-caption` 12px, `.mono`, `--muted-fg`.
- 더보기 버튼: `--t-sm` 14px, weight 600.
- 빈 상태 제목: `--t-base` 16px, weight 700 / 본문: `--t-sm` 14px, weight 400, `--muted-fg`.

**규칙:** 모든 숫자(동반율·실시간 등락률·종목코드·직접동반 N회)는 `.mono`(tabular-nums slashed-zero) 적용. 한글 라벨은 `--font-sans`.

## Color

`globals.css` oklch 토큰만 사용. 신규 토큰·하드코딩 hex 금지. Light(`:root`)/Dark(`.dark`) 두 모드 토큰이 자동 전환.

| Role | Value (Light / Dark) | Usage |
|------|----------------------|-------|
| Dominant (60%) | `--bg` `#FFFFFF` / `oklch(0.08 0 0)` | 페이지 배경 (섹션이 올라앉는 면) |
| Secondary (30%) | `--card` `#FFFFFF` / `oklch(0.12 0 0)` | 후보 행 카드, 더보기 버튼 배경 |
| Accent (10%) | `--accent` `oklch(0.95 0.03 250)` / `oklch(0.22 0.05 250)` (+ `--accent-fg`) · hover/focus 시 `--primary` `oklch(0.63 0.18 250)` / `oklch(0.72 0.16 250)` | 아래 "Accent reserved for" 목록 한정 |
| Destructive | `--destructive` (미사용) | 이 섹션에 파괴적 액션 없음 — 사용 안 함 |

**Accent reserved for (명시 — "all interactive" 절대 금지):**
1. **직접동반 근거 칩 fill** — `bg-[var(--accent)] text-[var(--accent-fg)]` (테마무관 co-surge 근거를 시각 구분, 실측 41% 포착).
2. **hover/focus border** — 행 카드·더보기 버튼 hover 시 `border-color: color-mix(in oklch, var(--primary) 30%, var(--border))`. 전역 double-ring focus 의 `--ring`(= `--primary` 계열).

그 외 어떤 요소도 accent/primary 채도색을 사용하지 않는다.

**Semantic (별개 축 — accent 아님, 트레이딩 정확성 핵심):**
- **실시간 등락률만** 한국 관례 색: 상승 `--up`(빨강 `oklch(0.66 0.20 22)`) / 하락 `--down`(파랑 `oklch(0.63 0.18 250)`) / 보합 `--flat`(`oklch(0.55 0 0)`).
- **강도 라인**(`.underbar`) = `--up`(동반 상승 강도).
- **후행형 배지** = `--down` 계열: border `color-mix(in oklch, var(--down) 35%, var(--border))` · text `--down` · bg `--down-bg`. (D+1 타이밍 신호 → down 톤. 단 의미는 "하락"이 아니라 "후행"이며 라벨 "후행형"으로 구분.)
- **공유 테마 칩 도트** = `--flat` (중립, `theme-chips` 시스템 테마 칩과 동일).

> **★ LOCKED 색 규칙:** 동반율(`conf_d0`) 및 모든 확률/비율 지표 = **중립 `--fg`**. 절대 빨강/파랑 금지 (등락이 아니므로 오해 유발). 실시간 등락률만 방향 색을 갖는다.

## Component Structure (실행 계약 — executor가 이 문서만으로 구현 가능해야 함)

### 배치 / 마운트
- 신규 클라이언트 컴포넌트 `StockComovementSection`(`'use client'`), `webapp/src/components/stock/` 하위.
- `stock-detail-client.tsx`에서 `<StockThemeChips stockCode={stock.code} />` **바로 다음**, 뉴스/토론 `<div className="space-y-6">` **앞**에 삽입. 부모 컨테이너는 `space-y-8` 스택.
- 루트 = `<section aria-label="동조 후보">`.

### 섹션 헤더 (StockThemeChips h2 캡션 패턴 그대로)
```
<h2 class="flex items-center gap-1.5 text-[length:var(--t-caption)] font-semibold
           uppercase tracking-wide text-[var(--muted-fg)]">
  <Waypoints class="size-3.5" aria-hidden /> 동조 후보
  <span class="ml-auto normal-case tracking-normal font-medium">급등 시 동반 ↑ · 동반율 순</span>  ← 선택
</h2>
```
- 아이콘 = lucide `Waypoints`(1순위) 또는 `Network`(대체). `size-3.5`(14px).
- 우측 힌트는 선택 — 좁은 폭에서 wrap/생략 가능.

### 후보 행 카드 (Variant C 레이아웃)
- 컨테이너: `position: relative; overflow: hidden`, `border border-[var(--border)] bg-[var(--card)] rounded-[var(--r)]`, padding `py-[var(--s-3)] px-[var(--s-4)]` (12/16, theme-rank-row 동일 — 목업 10/14를 4-scale로 정규화).
- **행 전체가 `/stocks/[code]` Link** (theme-rank-row 선례: `aria-label` + 전역 double-ring focus. `'use client'` 컴포넌트 내 `next/link`).
- hover: `border-color: color-mix(in oklch, var(--primary) 30%, var(--border))`, `transition-colors`.
- 그리드 2열 `grid-template-columns: 1fr auto`, `items-center`, gap `var(--s-3)`(12px).
- **좌측 컬럼** (`min-w-0 flex flex-col gap-[5px]` — 5px=칩/타이포 보정, hairline 예외):
  - 1줄(`flex items-baseline gap-2`): 종목명 `<b>`(700, `--t-base`, `--fg`) + 종목코드(`.mono`, `--t-caption`, `--muted-fg`).
  - 2줄(`flex flex-wrap items-center gap-2`): 근거 칩 그룹 + 후행형 배지(있을 때).
- **우측 메트릭** (`.metric` = `flex items-end gap-[var(--s-3)]` 12px):
  - 동반율: `.mono`, weight 800, `--t-h3`(20px), `line-height:1`, `text-right`, 색 = **중립 `--fg`**. 위에 미니라벨 "동반율"(10px/600/`--muted-fg`, margin-bottom 3px).
  - **co-surge 전용 후보(공유 테마 근거 없음, `sharedThemes=[]`)는 동반율 값 대신 "—"** (`.mono`, `--muted-fg`, 미니라벨 유지). conf_d0=0 을 "0%"로 표시하면 테마무관 동조(co-surge 경로의 존재 이유 — 실측 강한 동조의 41%)가 가장 약한 신호로 오해되므로 금지. 이 후보의 근거는 "직접동반 {N}회" 칩이 담당.
  - 실시간 등락률: `.mono`, weight 700, `--t-sm`(14px), `text-right`, `min-width: 62px`, 색 = 등락 방향(`--up`/`--down`/`--flat`). 위에 미니라벨 "실시간"(10px/600/`--muted-fg`).
- **강도 라인** (`.underbar`): `position: absolute; left: 0; bottom: 0; height: 3px; border-radius: 0 999px 999px 0; background: var(--up)`. `width = 결합점수% (min 4%, max 100%)`. `aria-hidden`.
  - 결합 점수 = `conf_d0 + lift + avg_ret + conf_d1` 기준(planner 가중치 확정). `lift`는 내부 디노이즈로만 사용(미표시).
  - 폭 계산식 = `Math.max(4, Math.min(100, strength * 100))` (theme-rank-row `barPct` 동형).

### 근거 칩 (D-03 — 둘 다 해당이면 둘 다 표시, **테마 먼저 → 직접동반**)
- **공유 테마 칩** (테마별 1개씩): `inline-flex items-center gap-[5px]`, outline `border border-[var(--border)]`, `rounded-full`, `px-[9px] py-px`, `--t-caption`, `--fg`. 앞에 flat 도트(`size-[5px] rounded-full bg-[var(--flat)]`) + 테마명. (`theme-chips` 시스템 테마 칩 스타일.)
- **직접동반 칩** (co-surge 근거 있을 때 1개): **accent fill** `bg-[var(--accent)] text-[var(--accent-fg)]`, `rounded-full`, `px-[9px] py-px`, `--t-caption`, weight 600. lucide `ArrowUpRight`(~11px, `gap-1`) + "직접동반 {N}회".

### 후행형 배지 (D-08 — D+1 후행 우세 후보)
- `inline-flex items-center gap-1`, `border` `color-mix(in oklch, var(--down) 35%, var(--border))`, `text-[var(--down)]`, `bg-[var(--down-bg)]`, `rounded-full`, `px-2`, 11px, weight 700. lucide `History`(또는 `Clock`, ~10px) + "후행형".

### 표시 동작 (★ 사용자 수정 — D-10 갱신: 기본 3 + 더보기)
- **초기 렌더 = 최대 3개**. `useState`로 `expanded` boolean 관리.
- 후보 **≤ 3개**면 더보기 버튼 **없음**.
- 후보 > 3개면 더보기 버튼 표시:
  - full-width, `border-[var(--border)] bg-[var(--card)] text-[var(--muted-fg)] rounded-[var(--r)]`, padding `py-[var(--s-2)]`(8px) `px-[var(--s-3)]`(12px), `--t-sm`, weight 600, `mt-[var(--s-2)]`.
  - hover: `text-[var(--fg)]` + `border-color: color-mix(in oklch, var(--primary) 30%, var(--border))`.
  - 라벨: 접힘 = "동조 후보 {N}개 더 보기" (N = 전체 − 3) / 펼침 = "접기".
- 펼침 시 서버가 반환한 전체 집합(기본 fetch `k=8`) 표시. (8 초과는 `?k=` 파라미터 — v1 더보기는 반환 집합 한도. 추가 fetch는 executor 재량.)

### 빈 상태 (D-06 — 두 경로 모두 빌 때만)
- 트리거: 공유 테마 없음 **AND** co-surge 이웃도 없음(= 후보 0개)일 때만.
- `border border-dashed border-[var(--border)] rounded-[var(--r-md)]`, `px-[var(--s-4)] py-[var(--s-5)]`(16/24), `text-center`.
- lucide `CircleOff`(옅은 circle-slash, ~22px, `--muted-fg`, margin-bottom 8px).
- 제목 "동조 데이터 부족"(`--t-base`, weight 700) + 본문(`--t-sm`, `--muted-fg`): "아직 이 종목과 함께 움직인 패턴을 찾지 못했습니다. 테마·동반 급등 이력이 쌓이면 표시됩니다."
