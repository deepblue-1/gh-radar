## Lessons Learned

- Claude Haiku 4.5 + `max_tokens=10` + 라벨 정확 매치 정책은 분류 task 에 충분 — temperature=0 으로 안정성 확보.
- p-limit(5) 동시성 한도는 Anthropic Tier 1 RPM 내 안전.
- ANTHROPIC_API_KEY 부재 시 classify 경로 graceful no-op — 로컬 dev/test 에서 500 에러 없이 기존 cycle 그대로 동작.
- Wave 기반 worktree 병렬은 `parallelization=false` 로도 sequential 실행 + 자동 병합 패턴이 안정적. 일부 worktree 는 수동 cleanup 필요(lock 잔존 이슈).
- Plan `autonomous: false` 플래그가 auto-mode 에서 "생성은 하되 실제 실행은 deferred" 경계를 명확히 해주어 안전하게 중단됨.
