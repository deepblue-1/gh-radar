### Auto-fixed Issues

**1. [Rule 1 - Bug] PLAN 본문 chart-colors.ts 코드와 W2 acceptance grep 자체 모순**

- **Found during:** Task 3 첫 verification
- **Issue:** PLAN 본문이 코드 예시에 JSDoc 헤더 4 곳 + inline trailing 주석 5 곳 (`// ≈ oklch(...)`) 으로 oklch 단어를 사용. W2 grep 패턴 `^[[:space:]]*[^/[:space:]].*oklch` 는 JSDoc 의 ` * ...` 라인도 매치 (`*` 가 `/` 가 아니라서) → 코드 라인 oklch literal 0건 검증 실패.
- **Fix:** JSDoc 헤더와 inline 주석을 모두 paraphrase. "CSS Color 4 함수 형식" 으로 단어 회피, hex 매핑 reference 는 외부 RESEARCH §Pitfall 9 로 정보 위임. 회귀 방지의 의도 (lightweight-charts 가 globals.css 토큰을 그대로 받지 못한다는 추적성) 는 JSDoc 첫 줄로 명시 보존.
