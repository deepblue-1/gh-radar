Config (from Plan 01): cfg.surgeThreshold (20), cfg.newsPerStock (5), cfg.surgeMax (80), cfg.classifyModel.

Data sources (server/src/routes/scanner.ts + quoteJoin patterns):
- stock_quotes: columns code, change_rate, price (change_rate numeric — filter .gte("change_rate", threshold))
- top_movers / stocks: code, name, market (name resolution)
- news_articles: columns id, stock_code, title, url, source, published_at, description; index (stock_code, published_at DESC); unique(stock_code, url)
- Pitfall 1: single .in() truncates at 1000 rows. Use per-stock top-K: fetch news chunked by code (QUOTE_CHUNK=200) then app-side keep top newsPerStock per code by published_at DESC.

Claude OUTPUT contract (system prompt enforces — indices only, no URL/title generation):
{ "themes":[{ "name":"초전도체", "reason":"...", "stockCodes":["294630","004920"], "newsRefs":[0,1] }],
  "singles":[{ "stockCode":"347700", "reason":"...", "newsRefs":[3] }] }

Anti-hallucination (D-04, RESEARCH §Pattern 3): input news lines are numbered globally (index N). Claude returns newsRefs=[indices]. Worker resolves index → actual news_articles {title,url,source}. Worker DROPS any stockCode not in the surge set, and any newsRef out of range. This mirrors discoverThemes resolveNamesToCodes philosophy.

Sort (D-05, worker-side NOT LLM): themes by stockCodes.length DESC, tie → member avg change_rate DESC. singles by change_rate DESC.
Classify (D-06): theme with <2 valid stockCodes → demote to single. Any single is 1 stock.

hash-skip clone-append (Pattern 4): fetch latest snapshot today (captured_at DESC limit 1). If prev.content_hash === hash → INSERT clone of prev.payload with is_carried=true (Claude call skipped). Else → clusterSurges (Claude 1x) → payload → INSERT is_carried=false. INSERT on PK conflict DO NOTHING (idempotent slot re-run) — use upsert with onConflict "trade_date,captured_at" ignoreDuplicates.

content_hash (Pattern 4 + Open Q3): SHA256 of sorted surge codes + news ids/count (NOT full titles — reduce over-sensitivity). Deterministic (sort before serialize).

captured_at slot: KST :30 slot of now. tradeDate = KST date. marketStatus: "closed" if slot >= 15:30 KST else "open".
