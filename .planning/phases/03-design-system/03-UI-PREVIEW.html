디자인 미리보기

이 페이지는 Phase 3 디자인 시스템 확정 전 시각 검토용입니다. BBAA 선택을 그대로 반영했습니다 — 데이터 밀집 톤 / 토스풍 빨강·파랑 / 순백·순검정 배경 / 16px 본문. 상단의 테마 전환 버튼으로 Light/Dark 를 비교하세요.

이 부분 바꾸고 싶으면 알려주세요 — 색상, 행 높이, 폰트 크기, 타이포 무엇이든 한 줄로 피드백 주시면 다음 라운드에서 즉시 반영합니다.
