    -- Phase 05.2: SCAN-04 trade_amount 컬럼 추가
    -- stocks.volume(주식수)과 별개로 거래대금(KRW)을 보관한다.
    -- KIS 등락률 순위 응답 acml_tr_pbmn 필드를 upsert한다.

      ADD COLUMN IF NOT EXISTS trade_amount bigint NOT NULL DEFAULT 0;

    규칙:
    - **반드시** `IF NOT EXISTS` 포함 — 재실행 idempotent 보장.
    - **`volume` 컬럼은 삭제하지 않음** (D-01). 스키마 상 두 컬럼이 공존한다.

    - 별도 backfill/UPDATE 문 없음 (D-02) — 기존 row는 DEFAULT 0으로 채워지고 다음 ingestion cycle(Wave 4 배포 후)에서 자동 덮어써진다.
    - 이 플랜은 **파일 작성만** — `supabase db push` 실행은 Wave 4 (Plan 05) 소관.
