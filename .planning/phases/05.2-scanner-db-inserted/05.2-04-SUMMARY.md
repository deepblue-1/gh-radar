# Phase 05.2 Plan 04: Webapp Scanner UI — 거래대금 + 서버 기준 갱신시각 Summary

## Decisions Made

- D-14: `stock.volume` 참조를 scanner 컴포넌트 디렉토리에서 완전 제거 (주석 포함). 컬럼 width 등 기존 디자인 그대로.
- D-15/D-16: 거래대금 포맷은 `formatTradeAmount` util 로 분리 → `<Number format='trade-amount'/>` 와 직접 import 양쪽에서 동일 결과.
- D-17/D-18: `usePolling` 의 `Date.now()` 갱신은 제거. `lastUpdatedAt` state 자체는 타입 호환을 위해 `undefined` 상수로 유지하고, 실제 값 전달은 `scanner-client` 가 fetcher 결과의 ISO → epoch ms 로 변환해 직접 prop 으로 전파.
- D-19: `ScannerFilters` 의 표시 포맷(`formatKstTime`) 과 prop 시그니처(`number | undefined`) 는 그대로 유지.
