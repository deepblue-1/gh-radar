```
tgt < 2000   → floor(tgt/1)*1
tgt < 5000   → floor(tgt/5)*5
tgt < 20000  → floor(tgt/10)*10
tgt < 50000  → floor(tgt/50)*50
tgt < 200000 → floor(tgt/100)*100
tgt < 500000 → floor(tgt/500)*500
else         → floor(tgt/1000)*1000
```
