## Sampling Rate

- **After every task commit:** Run `pnpm --filter @gh-radar/webapp test` (unit only, ~45s)
- **After every plan wave:** Run full suite (unit + e2e) — 약 3m
- **Before `/gsd-verify-work`:** Full suite must be green + 수동 체크 3항목 (아래 Manual-Only)
- **Max feedback latency:** 60s (unit), 240s (full suite)
