### Breakpoint 변경 전/후 요약

**전 (Phase 05 출시 상태):**
- `< 768px` (md 미만) → 모바일 카드
- `≥ 768px` (md 이상) → Table

**후 (Phase 06.2 D-23.1 통일):**
- `< 1024px` (lg 미만) → 인포그래픽 카드 (InfoStockCard)
- `≥ 1024px` (lg 이상) → Table

효과: iPad/태블릿 세로 모드 (768~1023px) 에서 가독성이 높은 카드 UI 가 노출되며, Watchlist 페이지(Plan 06) 와 동일한 기준으로 통일됨.
