- **KIWOOM_APPKEY/SECRETKEY required get()**: Phase 09.1 D-17 의 server 키움 호출이 production 필수 경로. test setup.ts 가 'test-kiwoom-appkey/secretkey' 주입. production 은 Plan 09 의 deploy-server.sh 가 GCP Secret Manager 에서 주입.

- **freshUpsert + cachedRow 합성에서 cached null 시 volume=0**: 신규 상장 + bootstrapMissingStocks 후 첫 cycle 직전 (stock_quotes row 부재) 경계 케이스. 이 시점 응답 volume=0 은 일시적 (다음 STEP1 cycle 에서 자연 갱신). UI 가 "거래량 0" 표시할 가능성 있으나 stub 이 아닌 정상 동작 (worker 와 server 의 시간 차이).
